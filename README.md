## 📖 About The Project

**AKTU Papers** is a student-driven initiative that provides **free access to semester-wise, branch-wise previous year question papers** of AKTU.  
We believe every student deserves quality exam preparation material without paying or signing up.

### Key Features

- ✅ **10,000+** students served
- ✅ **500+** question papers (2000–2025 sessions)
- ✅ **30+** branches (CS, EC, ME, CE, IT, EE, etc.)
- ✅ **Smart search** by subject code / name
- ✅ **No login required** – direct PDF download
- ✅ **Mobile friendly** & fast loading
- ✅ **DMCA compliant** – respects copyright
